# Module 5 Ticket API Backend

This project is a backend API service for a simple IT help desk ticket system. It is built with **Node.js**, **Express**, and **PostgreSQL**, and deployed on **Render**.

## Features

- Retrieve all tickets: `GET /tickets`
- Retrieve a ticket by ID: `GET /tickets/:id`
- Submit a contact message: `POST /contact`
- Connected to a remote PostgreSQL database
- Data seeded with 50 sample tickets
